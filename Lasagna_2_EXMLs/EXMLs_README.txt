----------------------------------------
Instructions for updating Lasagna biomes
----------------------------------------

1. Run PLUMGEN - https://github.com/SunnySummit/PLUMGEN (remember to check installation requirements)
2. For each folder, copy contents (subfolders NOT top folder) into '_BIOMES Exmls Folder Goes Here' (4x in total):

	1 XL 			- File > Bulk Import & Update
	2 Reg*			- ^same

	3 Spawner_Min 	- Click button, 'Import EXMLs/Merge Biomes', then export
	4 Spawner_Max 	- ^same

	*for BIOMES6 aka SpiffyBiomesPairs, use 'Near' distantobjects/LOD. for ALL others, use *default* settings

when prompted, manually type pak (can ignore lua) titles. should be obvious from current biome objects list, but:
